# Pizza Delivery App

A cross-platform food ordering application built with React Native and Expo.

## Features

- User Login
- Pizza Categories
- Product Catalog
- Product Details
- Responsive UI
- Cross-platform support (Android, iOS, Web)

## Tech Stack

- React Native
- Expo SDK 54
- React Navigation
- JavaScript

## Status

Currently being modernized from a legacy Expo Snack project.
